# PAY

謎盒逃脫是一個單頁面的解謎小遊戲，使用純 HTML 與 JavaScript 實作。玩家需要透過房間線索依序解開三道謎題，最後組合出四位數鎖碼。

## 玩法說明
1. 以瀏覽器開啟 `index.html`，無需額外建置或伺服器。
2. 在「房間線索」區收集提示，並觀察日誌中的紀錄。
3. 按順序完成三個謎題：
   - 星盤轉盤：依照壁畫提示點擊轉盤，取得第一碼。
   - 羽筆密碼：根據紙條的 Caesar 位移解碼出第二碼。
   - 符文速記：依符號對應數字輸入組合，得到第三碼。
4. 將取得的三段數字加上最後提示的「8」組成最終鎖碼並解鎖。

## 檔案結構
- `index.html`：完整的遊戲頁面、樣式與互動邏輯。
- `README.md`：專案與玩法概述。

## 進階提示
- 謎題進度可在左上角狀態面板查看，解鎖後進度條會更新。
- 日誌會紀錄每個動作與提示，若卡關可先回顧日誌內容。
